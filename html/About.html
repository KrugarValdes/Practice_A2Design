About GitHub Search

Введите логин или имя пользователя GitHub, которого вы хотите найти, и вам будут представлены результаты, соответствующие введенной строке. Для поиска пользователей используется GitHub Search API. Приложение после поиска выводит, первую найденную десятку пользователей GitHub. Некоторые предлагаемые поисковые запросы:

  • Ввод собственного логина(Например, мой - KrugarValdes),
  • Ввод имени человека(Например, Bill Gates),
  • Ввод разных слов для проверки приложения.
В стороку поиска можно писать прописными буквами, на результат это не повлияет.

Поиск

Введите любой текст для поиска пользователй GitHub. Поиск начнется сразу же после ввода данных, результатом будет вывод 10 пользователе(Аватар, логин, ссылка на профиль GitHub) с самым похожим логином и именем указанным в профиле.

Если поиск не возвращает ожидаемых результатов или вообще не дает результатов, подумайте о том, чтобы переписать запрос.

При обработке ввода в приложении используется функция debounce, для снижения количество запросов. Приложение адаптировано для экранов разного разрешения.
